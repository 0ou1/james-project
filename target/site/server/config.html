Configure Apache James Server

All configuration files resides in the ./conf and ./conf/META-INF folder.

With default Spring wiring, we ship with default configuration. It gets automatically deployed in the conf folder while unzipping. Of course, you can change the files *.xml the settings to match your needs.

For Guice based wiring, you are encouraged to take a look at default configuration of the James docker images.

Please note that the configuration is dependent of the wiring you have chosen. Some configuration files are specific to some back-ends, to some protocols. Please also note that some configuration files are not required for Guice. In this case, it will be specified.

The following sections presents configuration files that could be used for the various wiring available

Note that viable defaults are provided.

Config File Sample Explanation Page Config Comment Default behaviour
domainlist.xml Domain List Configuration
batchsizes.properties Mailbox specific limits to avoid that a single request overwhelm the server Cassandra Guice only
cassandra.properties Cassandra Configuration Tries to connect to the cassandra host. Start fails if no viable host can be found.
usersrepository.xml Users Configuration
recipientrewritetable.xml Recipient Rewrite Table Configuration
listeners.xml Additional mailbox listeners Guice specific feature No additional listeners

lmtpserver.xml LMTP Configuration No LMTP server enabled
pop3server.xml POP3 Configuration No POP3 server enabled
imapserver.xml IMAP4 Configuration No IMAP server enabled
jmap.properties JMAP configuration No JMAP server enabled
tika.properties Tika configuration Tries to connect to the tika host. If none, text extraction will fail, but server will behave normally.
managesieveserver.xml MANAGESIEVE Configuration Experimental. May need bug fixing. Contributions are welcomed. No MANAGESIEVE server enabled.
mailetcontainer.xml Mailet Container Configuration Provides a default mailet container system.
jmx.properties
 JMX Configuration
webadmin.properties
 Webadmin Configuration
